# Todo_List
This app lets you list everything that you have to do with the prioritized tasks on the top and least ones to the bottom.

**Technology Stack**

<ul>
  <li> HTML
  <li> CSS  
  <li> Express
  <li> Nodejs
  <li> Mongodb  
</ul>
